## Document Categories

For the training pipeline, all source documents map to these voice registers:

| Category | Description | Voice Register |
|----------|-------------|----------------|
| **federalist** | Federalist Papers | Polished argumentative prose |
| **essays** | Political essays, pamphlets, memoranda | Formal analytical writing |
| **legislative** | Resolutions, reports, memorials, plans | Institutional/legal drafting |
| **speeches** | Congressional, Convention, VA legislature | Oral/combative/extemporaneous |
| **presidential** | Inaugurals, messages, vetoes, proclamations | Executive authority |
| **correspondence** | Letters to individuals | Private intellectual voice |
| **constitutional-analysis** | Observations, Q&A, convention notes | Analytical/explanatory |
| **retirement** | Late-career essays, slavery Q&A, autobiography | Reflective/retrospective |
